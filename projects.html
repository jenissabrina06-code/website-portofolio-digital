Android App

Aplikasi Android ini dirancang untuk mengumpulkan input pilihan pengguna melalui formulir sederhana yang terdiri dari beberapa opsi menggunakan radio button. Pengguna dapat memilih kondisi tertentu, seperti cuaca dan ketersediaan uang, lalu mengonfirmasi pilihannya melalui tombol yang disediakan. Aplikasi ini dikembangkan sebagai sarana pembelajaran untuk memahami konsep input pengguna, pengolahan data, serta interaksi antarmuka pada pengembangan aplikasi Android.

  • Teknologi: Android Studio, Kotlin
  • Peran: Pengembang Aplikasi Android

Website e-commerce

Website e-commerce ini dikembangkan sebagai sistem informasi penjualan produk hewan peliharaan yang memungkinkan pengguna untuk mengakses informasi ketersediaan produk, seperti makanan hewan, vitamin, mainan, kandang, serta perlengkapan perawatan lainnya secara online. Sistem ini menyediakan fitur pencarian produk, tampilan detail produk, dan proses pemesanan yang terstruktur. Dengan adanya website ini, proses transaksi jual beli menjadi lebih efisien, transparan, dan mudah diakses kapan saja tanpa harus datang langsung ke toko fisik.

  • Teknologi: HTML, CSS, JavaScript
  • Peran: Pengembang Front-End

Desain Input Barang

Aplikasi Input Barang merupakan aplikasi desktop sederhana yang digunakan untuk mengelola data barang, seperti kode barang, nama barang, harga, satuan, dan stok. Aplikasi ini membantu pengguna dalam melakukan proses pencatatan, penyimpanan, pengeditan, dan penghapusan data barang secara terstruktur melalui antarmuka grafis.

  • Teknologi: Java, Java Swing, MySQL
  • Peran: Pengembang Aplikasi Desktop
